DOMNode::hasChildNodes

(PHP 5)

DOMNode::hasChildNodes — Checks if node has children

Description

public bool DOMNode::hasChildNodes ( void )

This function checks if the node has children.

Return Values

Returns TRUE on success or FALSE on failure.
