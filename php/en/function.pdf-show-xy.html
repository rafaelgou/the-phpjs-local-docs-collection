PDF_show_xy

(PHP 4, PECL pdflib >= 1.0.0)

PDF_show_xy — Output text at given position

Description

bool PDF_show_xy ( resource $p , string $text , float $x , float $y )

Prints text in the current font. Returns TRUE on success or FALSE on failure.
