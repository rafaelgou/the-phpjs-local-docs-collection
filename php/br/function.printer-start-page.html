printer_start_page

(PECL printer SVN)

printer_start_page — Começa uma nova página

Descrição

bool printer_start_page ( resource $printer_handle )

A função cria uma nova página no documento ativo. Para um exemplo veja printer_start_doc().

Parâmetros

printer_handle

printer_handle precisa ser um válido manipulador da printer.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
