classkit_import

(PECL classkit >= 0.3)

classkit_import — Importa métodos definidos em outra classe

Descrição

array classkit_import ( string $filename )

Nota: This function cannot be used to manipulate the currently running (or chained) method.

Aviso

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome, incluindo toda documentação pode ser modificado sem aviso em futuras versões do PHP. Esta função deve ser usada por sua própria conta e risco.

Parâmetros

filename

Nome do arquivo onde foi definido o método a ser importado

Valor Retornado

Array contendo os métodos importados

Exemplos

Exemplo #1 classkit_import() example

<?php
// file: newclass.php
class Example {
    function foo() {
        return "bar!\n";
    }
}
?>
<?php
// requires newclass.php (see above)
class Example {
    function foo() {
        return "foo!\n";
    }
}

$e = new Example();

// output original
echo $e->foo();

// import replacement method
classkit_import('newclass.php');

// output imported
echo $e->foo();

?>

O exemplo acima irá imprimir:

foo!
bar!
