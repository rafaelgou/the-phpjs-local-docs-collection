iterator_count

(PHP 5 >= 5.1.0)

iterator_count — Conta o número de elementos do iterador

Descrição

int iterator_count ( IteratorAggregate $iterator )
Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Conta os elementos em um iterador.

Parâmetros

iterator

O iterador a ser contado.

Valor Retornado

O número de elementos no iterator.
