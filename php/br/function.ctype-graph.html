ctype_graph

(PHP 4 >= 4.0.4, PHP 5)

ctype_graph — Verifica se os caracteres são imprimíveis exceto espaço

Descrição

bool ctype_graph ( string $text )

Verifica se todos os caracteres na string fornecida, text, são visíveis.

Parâmetros

text

A string a ser testada.

Valor Retornado

Retorna TRUE se todos caracteres em text são imprimíveis e criam uma saída visível (sem white space), FALSE caso contrário.

Exemplos

Exemplo #1 A ctype_graph() example

<?php
$strings = array('string1' => "asdf\n\r\t", 'string2' => 'arf12', 'string3' => 'LKA#@%.54');
foreach ($strings as $name => $testcase) {
    if (ctype_graph($testcase)) {
        echo "The string '$name' consists of all (visibly) printable characters.\n";
    } else {
        echo "The string '$name' does not consist of all (visibly) printable characters.\n";
    }
}
?>

O exemplo acima irá imprimir:

The string 'string1' does not consist of all (visibly) printable characters.
The string 'string2' consists of all (visibly) printable characters.
The string 'string3' consists of all (visibly) printable characters.
