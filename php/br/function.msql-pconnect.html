msql_pconnect

(PHP 4, PHP 5)

msql_pconnect — Abre uma conexão mSQL persistente

Descrição

int msql_pconnect ([ string $server [, string $username [, string $password ]]] )

msql_pconnect() funciona muito parecido com msql_connect() com duas diferenças maiores.

Primeiro, ao conectar, a função irá primeiro tentar encontrar uma conexão persistente que já esteja aberta com o mesmo servidor. Se encontrar uma, um identificador para esta conexão será retornado ao invés de criar uma nova conexão.

Segundo, a conexão com o servidor SQL não será fechada ao final da execução do script. Ao invés, a conexão permanecerá aberta para uso futuro ( msql_close() não irá fechar conexões persistentes abertas por msql_pconnect()).

Retorna um identificador de conexão mSQL persistente em caso de sucesso, ou FALSE em caso de erro.
