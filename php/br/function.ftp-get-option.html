ftp_get_option

(PHP 4 >= 4.2.0, PHP 5)

ftp_get_option — Obtém várias opções em tempo de execução da conexão FTP atual

Descrição

mixed ftp_get_option ( resource $ftp_stream , int $option )

Esta função retorna o valor para a option solicitada da conexão FTP especificada.

Parâmetros

ftp_stream

O identificador para a conexão FTP.

option

Atualmente, as seguintes opções são suportadas:

Opções em tempo de execução do FTP suportadas
FTP_TIMEOUT_SEC Retorna o limite de tempo usado atualmente para as operações de rede.
FTP_AUTOSEEK Retorna TRUE se esta opção estiver ativa, FALSE se não.

Valor Retornado

Retorna o valor em caso de sucesso ou FALSE se a option indicada não for suportada, neste caso, uma mensagem de aviso também é emitida.

Exemplos

Exemplo #1 Exemplo ftp_get_option()

<?php
// Get the timeout of the given FTP stream
$timeout = ftp_get_option($conn_id, FTP_TIMEOUT_SEC);
?>
