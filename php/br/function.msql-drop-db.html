msql_drop_db

(PHP 4, PHP 5)

msql_drop_db — Exclui um banco de dados mSQL

Descrição

bool msql_drop_db ( string $database_name [, resource $link_identifier ] )

msql_drop_db() tenta remover um banco de dados do servidor mSQL.

Parâmetros

database_name

O nome do banco de dados.

link_identifier

A conexão mSQL. Se não for especificado, a ultima conexão aberta por msql_connect() é assumida. Se uma conexão assim não for encontrada, a função irá tentar establecer uma conexão como se a função msql_connect() fosse chamada, e irá usa-la.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
