ibase_blob_open

(PHP 5)

ibase_blob_open — Abre um blob para obter partes de dados

Descrição

resource ibase_blob_open ( resource $link_identifier , string $blob_id )
resource ibase_blob_open ( string $blob_id )

Abre um BLOB existente para leitura.

Parâmetros

link_identifier

Um link identificador InterBase. Se omitido, o último link aberto é usado.

blob_id

Um id de BLOB.

Valor Retornado

Retorna um manipulador de BLOB para uso posterior com ibase_blob_get() ou FALSE em caso de falha.

Veja Também

  • ibase_blob_close() - Fecha o blob
  • ibase_blob_echo() - Envia o conteúdo do blob para o browser
  • ibase_blob_get() - Retorna uma quantidade especificada de bytes de dados de um blob aberto
