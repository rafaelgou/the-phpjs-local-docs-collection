sqlite_has_more

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_has_more — Retorna quando existem ou não mais linhas disponíveis

Descrição

bool sqlite_has_more ( resource $result )

Verifica se mais linhas estão disponíveis para um dado conjunto de resultados.

Parâmetros

result

O resource de resultado do SQLite.

Valor Retornado

Retorna TRUE se há mais linhas disponíveis para o manipultador result, ou FALSE caso contrário.

Veja Também

  • sqlite_num_rows() - Retorna o número de linhas em um conjunto de resultados.
  • sqlite_changes() - Retorna o número de linhas que foram alteradas pela SQL mais recente.
