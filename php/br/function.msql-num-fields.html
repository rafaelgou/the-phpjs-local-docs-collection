msql_num_fields

(PHP 4, PHP 5)

msql_num_fields — Retorna o número de campos em um resultado

Descrição

int msql_num_fields ( resource $result )

msql_num_fields() retorna o número de campos em um conjunto de resultados.

Parâmetros

result

O resulatdo resource que esta sendo avaliado. Este resultado vem de uma chamada a msql_query().

Valor Retornado

Retorna o número de campos em um conjunto de resultados.

  • msql_fetch_field() - Retorna informação sobre um campo
  • msql_num_rows() - Retorna o número de registros em um resultado
