mysqli::more_results

mysqli_more_results

(PHP 5)

mysqli::more_results -- mysqli_more_results — Verifica se há mais algum resultado de uma multi query

Descrição

bool mysqli::more_results ( void )
bool mysqli_more_results ( mysqli $link )

Indica se um ou mais conjunto de resultados estão disponíveis de uma chamada anterior a mysqli_multi_query().

Parâmetros

link

Procedural style only: A link identifier returned by mysqli_connect() or mysqli_init()

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplos

Veja mysqli_multi_query().

Veja Também

  • mysqli_multi_query() - Performs a query on the database
  • mysqli_next_result() - Prepara o próximo resultado de multi_query
  • mysqli_store_result() - Transfers a result set from the last query
  • mysqli_use_result() - Initiate a result set retrieval
