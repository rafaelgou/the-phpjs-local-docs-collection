imagepsfreefont

(PHP 4, PHP 5)

imagepsfreefont — Libera a memória usada por uma fonte PostScript Type

Descrição

bool imagepsfreefont ( resource $fontindex )

imagepsfreefont() libera a memória usada por uma fonte PostScript Type 1.

Parâmetros

fontindex

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Notas

Nota: Esta função somente está disponível se o PHP estiver compilado com --with-t1lib[=DIR] .
