chunk_split

(PHP 4, PHP 5)

chunk_split — Divide uma string em pequenos pedaços

Descrição

string chunk_split ( string $body [, int $chunklen [, string $end ]] )

Pode dividir uma string em pequenos pedaços que são úteis para ex. converter a saída de base64_encode() para semânticas RFC 2045 . Ela insere end a cada chunklen caracteres.

Parâmetros

body

A string a ser dividida.

chunklen

O tamanho do pedaço. Padrão para 76.

end

A seqüência de terminação. Padrão para "\r\n".

Valor Retornado

Retorna a string dividida.

Exemplos

Exemplo #1 Exemplo da chunk_split()

<?php
// format $data using RFC 2045 semantics
$new_string = chunk_split(base64_encode($data));
?>
