disk_free_space

(PHP 4 >= 4.1.0, PHP 5)

disk_free_space — Retorna o espaço disponível no diretório

Descrição

float disk_free_space ( string $directory )

Dado uma string contendo um diretório, esta função retornará o numero de bytes disponíveis no sistema de arquivos ou partição de disco.

Parâmetros

directory

Um diretório do sistema de arquivo ou partição de disco.

Nota:

Dado um nome de arquivo ao invés de um diretório, o comportamento da função não é especificado a pode diferir entre sistemas operacionais e versões do PHP.

Valor Retornado

Retorna o número de bytes disponíveis como um float.

Exemplos

Exemplo #1 Exemplo da disk_free_space()

<?php
// $df contém o número de bytes disponível em "/"
$df = disk_free_space("/");
?>

Notas

Nota: Esta função não trabalha com arquivos remotos, de forma que o arquivo a ser examinado precisa ser acessível pelo sistema de arquivos do servidor.
