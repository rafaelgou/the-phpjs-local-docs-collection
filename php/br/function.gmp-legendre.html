gmp_legendre

(PHP 4 >= 4.0.4, PHP 5)

gmp_legendre — Símbolo de Legendre

Descrição

int gmp_legendre ( resource $a , resource $p )

Computa o »  Símbolo de Legendre de a e p. p deve ser inteiro e positivo.

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

p

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Precisa ser ímpar e positivo.

Valor Retornado

Um resource de um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_legendre()

<?php
echo gmp_legendre("1", "3") . "\n";
echo gmp_legendre("2", "3") . "\n";
?>
