fam_resume_monitor

(PHP 5 <= 5.0.5)

fam_resume_monitor — Recomeça um monitoramento suspendido

Descrição

bool fam_resume_monitor ( resource $fam , resource $fam_monitor )

Recomeça o monitoramento de um recurso previamente suspendido usando fam_suspend_monitor().

Parâmetros

fam

Um resource representando a conexão para um serviço FAM retornado pela fam_open()

fam_monitor

Um resource retornado por uma das funções fam_monitor_XXX

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
