decbin

(PHP 4, PHP 5)

decbin — Decimal para binário

Descrição

string decbin ( int $number )

Retorna uma string contendo a representação binária do parâmetro number. O maior número que pode ser convertido é 4294967295 em decimal, resultando em uma strings de 32 números 1.

Parâmetros

number

Valor decimal para converter

Valor Retornado

String binária representando o number

Exemplos

Exemplo #1 Exemplo da decbin()

<?php
echo decbin(12) . "\n";
echo decbin(26);
?>

O exemplo acima irá imprimir:

1100
11010
