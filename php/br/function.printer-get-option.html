printer_get_option

(PECL printer SVN)

printer_get_option — Obtém dados de configuração da impressora

Descrição

mixed printer_get_option ( resource $handle , string $option )

A função obtém as definições de configuração de option. handle deve ser um manipulador válido para a impressora. De uma olhada em printer_set_option() para as definições que podem ser obtidas, adicionalmente as seguintes definições podem ser obtidas:

  • PRINTER_DEVICENAME retorna o devicename da impressora.
  • PRINTER_DRIVERVERSION retorna a versão do driver da impressora.

Exemplo #1 Exemplo printer_get_option()

<?php
$handle = printer_open();
echo printer_get_option($handle, PRINTER_DRIVERVERSION); 
printer_close($handle);
?>
