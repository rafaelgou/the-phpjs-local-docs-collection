ftp_systype

(PHP 4, PHP 5)

ftp_systype — Retorna o identificador do tipo do sistema do servidor FTP remoto

Descrição

string ftp_systype ( resource $ftp_stream )

Retorna o identificador do tipo do sistema do servidor FTP remoto.

Parâmetros

ftp_stream

O identificador com a conexão FTP.

Valor Retornado

Retorna o tipo do sistema remoto, ou FALSE em caso de erro.

Exemplos

Exemplo #1 Exemplo ftp_systype()

<?php

// ftp connection
$ftp = ftp_connect('ftp.example.com');
ftp_login($ftp, 'user', 'password');

// get the system type
if ($type = ftp_systype($ftp)) {
    echo "Example.com is powered by $type\n";
} else {
    echo "Couldn't get the systype";
}

?>

O exemplo acima irá imprimir algo similar à:

Example.com is powered by UNIX
