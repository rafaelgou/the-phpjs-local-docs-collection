bcdiv

(PHP 4, PHP 5)

bcdiv — Dividir dois números de precisão arbitrária

Descrição

string bcdiv ( string $left_operand , string $right_operand [, int $scale ] )

Divide o left_operand pelo right_operand e retorna o resultado. O parametro opcional scale.

Parâmetros

left_operand

O operando da esquerda, como uma string.

right_operand

O operando da direita, como uma string.

scale

Este parâmetro opcional é usado para definir o número de digitos depois do ponto decimal no resultado. Você pode definir a escala padrão global para todas as funções usando bcscale().

Valor Retornado

Retorna o resultado da divisão como uma string, ou NULL se right_operand é 0.

Exemplos

Exemplo #1 Exemplo da bcdiv()

<?php

echo bcdiv('105', '6.55957', 3);  // 16.007

?>
