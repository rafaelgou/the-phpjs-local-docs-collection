imagerectangle

(PHP 4, PHP 5)

imagerectangle — Desenha um retângulo

Descrição

int imagerectangle ( resource $image , int $x1 , int $y1 , int $x2 , int $y2 , int $col )

imagerectangle() cria um retângulo de cor col na imagem image começando na coordenada superior esquerda x1, y1 e terminando na coordenada inferior direita x2, y2. 0, 0 é o canto superior esquerdo da imagem.
