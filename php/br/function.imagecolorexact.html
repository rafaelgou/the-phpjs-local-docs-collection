imagecolorexact

(PHP 4, PHP 5)

imagecolorexact — Retorna o índice da cor especificada

Descrição

int imagecolorexact ( resource $image , int $red , int $green , int $blue )

Retorna o índice da cor especificada na paleta da imagem.

Se a cor não existir na paleta de imagem, é retornado -1.

Se você criou a imagem a partir de um arquivo, apenas as cores usadas na imagem são resolvidas. Cores presentes apenas na paleta não são resolvidas.

Veja também imagecolorclosest().
