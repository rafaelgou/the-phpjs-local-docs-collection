floatval

(PHP 4 >= 4.2.0, PHP 5)

floatval — Retorna o valor float da variável

Descrição

float floatval ( mixed $var )

Obtém o valor float de var.

Parâmetros

var

Pode ser qualquer tipo escalar. Você não pode usar floatval() em arrays ou objetos.

Valor Retornado

O valor float de uma dada variável.

Exemplos

Exemplo #1 Exemplo da floatval()

<?php
$variavel = '122.34343The';
$valor_float = floatval ($variavel);
echo $valor_float; // mostra 122.34343
?>
