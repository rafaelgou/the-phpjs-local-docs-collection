base64_decode

(PHP 4, PHP 5)

base64_decode — Decodifica dados codificados com MIME base64

Descrição

string base64_decode ( string $data [, bool $strict ] )

Decodifica uma informação base64 data.

Parâmetros

data

A informação a ser decodificada.

strict

Retorna FALSE se a entrada contém espaço ou algum outro separador.

Valor Retornado

Retorna a informação original ou FALSE em falha. O dado retornado pode ser binário.

Changelog

Versão Descrição
5.2.0 Adicionado strict

Exemplos

Exemplo #1 Exemplo da base64_decode()

<?php
$str = 'VGhpcyBpcyBhbiBlbmNvZGVkIHN0cmluZw==';
echo base64_decode($str);
?>

O exemplo acima irá imprimir:

This is an encoded string
