printer_end_page

(PECL printer SVN)

printer_end_page — Fecha a página ativa

Descrição

bool printer_end_page ( resource $handle )

A função fecha a página ativa no documento ativo. Para um exemplo veja printer_start_doc(). handle deve ser um manipulador válido para a impressora.
