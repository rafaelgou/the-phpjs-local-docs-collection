xml_get_error_code

(PHP 4, PHP 5)

xml_get_error_code — Obtém um código de erro do analisador XML

Descrição

int xml_get_error_code ( resource $parser )

Obtém o código de erro do analisador XML.

Parâmetros

parser

Uma referência para o analisador XML obter código de erro.

Valor Retornado

Esta função retorna FALSE se parser não se refere a um analisadore válido, senão ele retorna um dos códigos de erro listados na seção código de erros.
