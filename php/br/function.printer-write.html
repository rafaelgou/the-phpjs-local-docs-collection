printer_write

(PECL printer SVN)

printer_write — Escreve daos na impressora

Descrição

bool printer_write ( resource $handle , string $content )

Escreve content diretamente para a impressora. Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

handle deve ser um manipulador válido para a impressora.

Exemplo #1 Exemplo printer_write()

<?php
$handle = printer_open();
printer_write($handle, "Text to print");
printer_close($handle);
?>
