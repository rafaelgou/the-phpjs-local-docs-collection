array_change_key_case

(PHP 4 >= 4.2.0, PHP 5)

array_change_key_case — Modifica todas as chaves em um array

Descrição

array array_change_key_case ( array $input [, int $case ] )

Retorna um array com todas as chaves de input em minúsculo ou maiúsculo. Índice numéricos são mantidos como são.

Parâmetros

input

O array para modificar

case

Pode ser CASE_UPPER ou CASE_LOWER (padrão)

Valor Retornado

Retorna um array com as chaves em minúsculo ou maiúsculo, ou false se input não for um array.

Erros

Dispara E_WARNING se input não for um array.

Exemplos

Exemplo #1 Exemplo da array_change_key_case()

<?php
$input_array = array("primeiRo" => 1, "segunDo" => 4);
print_r(array_change_key_case($input_array, CASE_UPPER);
?>

O exemplo acima irá imprimir:

Array
(
    [PRIMEIRO] => 1
    [SEGUNDO] => 2
)

Notas

Nota:

Se uma matriz tem índices que serão o mesmo após a utilização desta função (ex: "chaVE" e "CHave"), o valor que estiver mais no fim na matriz irá sobrescrever os outros índices.
