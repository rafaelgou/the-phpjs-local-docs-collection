dcgettext

(PHP 4, PHP 5)

dcgettext — Substitui o domínio por um lookup

Descrição

string dcgettext ( string $domain , string $message , int $category )

Esta função permite que você sobreescreva o domínio atual por uma simples mensagem de lookup.

Parâmetros

domain

O domínio

message

A mensagem

category

A categoria

Valor Retornado

Uma string em sucesso.
