ibase_commit

(PHP 5)

ibase_commit — Grava uma transação

Descrição

bool ibase_commit ([ resource $link_or_trans_identifier ] )

Grava uma transação.

Parâmetros

link_or_trans_identifier

Se for chamada sem argumento, esta função grava a transação padrão na conexão padrão. Se o argumento é um identificador de conexão, a transação padrão na conexão correspondente será gravada. Se o argumento é um identificador de transação, a transação correspondente será gravada.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
