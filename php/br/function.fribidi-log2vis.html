fribidi_log2vis

(PHP 4 >= 4.0.4 and PHP 4 <= 4.1.0, PECL fribidi >= 1.0)

fribidi_log2vis — Converte uma string lógica para uma visual

Descrição

string fribidi_log2vis ( string $str , string $direction , int $charset )

Converte uma string lógica para uma visual.

Parâmetros

str

A string lógica.

direction

Uma de FRIBIDI_RTL, FRIBIDI_LTR or FRIBIDI_AUTO.

charset

Uma das constantes FRIBIDI_CHARSET_XXX.

Valor Retornado

Retorna a string visual em caso de sucesso FALSE em caso de falha.
