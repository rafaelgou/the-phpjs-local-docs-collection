similar_text

(PHP 4, PHP 5)

similar_text — Calcula a similaridade entre duas strings

Descrição

int similar_text ( string $first , string $second [, float &$percent ] )

Calcula a similaridade entre duas strings como descrito em Oliver [1993]. Note que esta implementação não usa um stack como no pseudo-código de Oliver, mas chamadas recursivas as quais podem ou não tornar todo o processo mais rapido. Note também que a complexidade deste algotimo é O(N**3) aonde N é o tamanho da maior string.

Parâmetros

primeira

A primeira string.

segunda

A segunda string.

percentual

Passando uma referencia como terceiro argumento, a função similar_text() irá calcular o percentual de similaridade para você.

Valor Retornado

Retorna o número de caracteres iguais em ambas as strings.
