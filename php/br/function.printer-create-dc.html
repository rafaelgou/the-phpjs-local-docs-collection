printer_create_dc

(PECL printer SVN)

printer_create_dc — Cria um novo new device context

Descrição

void printer_create_dc ( resource $handle )

A função cria um novo. Um device context é usado para customizar objetos gráficos do documento. handle deve ser um manipulador válido para uma impressora.

Exemplo #1 Exemplo printer_create_dc()

<?php
$handle = printer_open();
printer_start_doc($handle);
printer_start_page($handle);

printer_create_dc($handle);
/* do some stuff with the dc */
printer_set_option($handle, PRINTER_TEXT_COLOR, "333333");
printer_draw_text($handle, 1, 1, "text");
printer_delete_dc($handle);

/* create another dc */
printer_create_dc($handle);
printer_set_option($handle, PRINTER_TEXT_COLOR, "000000");
printer_draw_text($handle, 1, 1, "text");
/* do some stuff with the dc */

printer_delete_dc($handle);

printer_end_page($handle);
printer_end_doc($handle);
printer_close($handle);
?>
