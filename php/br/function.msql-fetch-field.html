msql_fetch_field

(PHP 4, PHP 5)

msql_fetch_field — Retorna informação sobre um campo

Descrição

object msql_fetch_field ( resource $query_identifier , int $field_offset )

Retorna um objeto contendo informações do campo

msql_fetch_field() pode ser usado para obter informações sobre os campos no resultado de uma consulta. se o número do campo não for indicado, o próximo campo que não tenha sido obtido por msql_fetch_field() será o campo retornado.

As propriedades do objeto são:

  • name - nome da coluna
  • table - nome da tabela a qual esta coluna pertence
  • not_null - 1 se a coluna não puder ser NULL
  • primary_key - 1 se a coluna é a chave primária
  • unique - 1 se a coluna é chave única
  • type - o tipo da coluna

Veja também msql_field_seek().
