rename

(PHP 4, PHP 5)

rename — Renomeia um arquivo ou diretório

Descrição

bool rename ( string $oldname , string $newname [, resource $context ] )

Tenta renomear oldname para newname.

Parâmetros

oldname

Nota:

O nome antigo. O wrapper usado em oldname precisa combinar com o wrapper usado em newname.

newname

O novo nome.

context

Nota: O suporte ao contexto foi adicionado no PHP 5.0.0. Para uma descrição de contextos, veja Streams.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Changelog

Versão Descrição
5.0.0 rename() pode agora também ser usada com algum URL wrappers. Consulte Supported Protocols and Wrappers para uma lista de wrappers que rename() suporta.
4.3.3 rename() agora pode renomear arquivos de diferente partições em sistemas baseados em *nix.

Exemplos

Exemplo #1 Exemplo da rename()

<?php
  rename("/tmp/tmp_file.txt", "/home/user/login/docs/my_file.txt");
?>
