filepro_fieldwidth

(PHP 4, PHP 5 <= 5.1.6)

filepro_fieldwidth — Obtém a largura de um campo

Descrição

int filepro_fieldwidth ( int $field_number )

Retorna a largura do campo correspondente a field_number.

Parâmetros

field_number

O número do campo.

Valor Retornado

Retorna a largura do campo como um inteiro, ou FALSE e, caso de erro.
