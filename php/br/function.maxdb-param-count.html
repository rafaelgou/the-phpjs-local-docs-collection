maxdb_param_count

(PECL maxdb 1.0)

maxdb_param_count — Sinônimo de maxdb_stmt_param_count()

Descrição

Esta função é um apelido para: maxdb_stmt_param_count().

Esta função é um apelido obsoleto e apenas existe para razão em manter a compatibilidade com versões anteriores. É recomendado não usar esta função já que um dia ela deve ser removida do PHP.
