dbase_open

(PHP 4, PHP 5)

dbase_open — Abre um banco de dados

Descrição

int dbase_open ( string $filename , int $mode )

dbase_open() abre um banco de dados dBase com o modo de acesso dado.

Nota: Quando o safe-mode está ativo, o PHP verifica se os arquivo(s) ou diretórios que estão usados na operação tem o mesmo UID (proprietário) do script que está sendo executado.

Nota:

Esta função é afetada por open_basedir.

Parâmetros

filename

O nome do banco de dados. Pode ser um caminho relativo ou absoluto para o arquivo onde o dBase irá guardar os seus dados.

mode

Um inteiro que corresponda a chamada de sistema open() (Tipicamente 0 significa somente-leitura, 1 apenas para escrita e 2 significa leitura e escrita).

Nota:

VoC~e não pode abrir um banco de dados dBase em modo somente para escrita já que a função irá falhar ao ler os cabeçalhos com informações e então você não pode usar 1 como o parâmetro mode.

Exemplos

Exemplo #1 Abrindo um banco de dados dBase

<?php

// open in read-only mode
$db = dbase_open('/tmp/test.dbf', 0);

if ($db) {
  // read some data ..

  dbase_close($db);
}

?>

Valor Retornado

Retorna um identificador do banco de dados se for aberto com sucesso, ou FALSE se um erro aconteceu.
