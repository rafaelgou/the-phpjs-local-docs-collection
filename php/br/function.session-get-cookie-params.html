session_get_cookie_params

(PHP 4, PHP 5)

session_get_cookie_params — Obtém os parâmetros do cookie da sessão

Descrição

array session_get_cookie_params ( void )

Obtém parâmetros do cookie da sessão.

Valor Retornado

Retorna um array com informação do cookie da sessão atual, o array contém os seguintes items:

  • "lifetime" - A duração do cookie em segundos.
  • "path" - O caminho onde a informação está guardada.
  • "domain" - O domínio do cookie.
  • "secure" - O cookie seria enviado apenas sobre conexões seguras.
  • "httponly" - O cookie somente pode ser acessado atráves do protocolo HTTP.

Changelog

Versão Descrição
5.2.0 A entrada "httponly" foi adicionada no array retornado.
4.0.4 A entrada "secure" foi adicionada no array retornado.
