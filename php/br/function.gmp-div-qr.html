gmp_div_qr

(PHP 4 >= 4.0.4, PHP 5)

gmp_div_qr — Divide números e retorna o quociente e o resto da divisão

Descrição

array gmp_div_qr ( resource $n , resource $d [, int $round ] )

Esta função divide n por d.

Parâmetros

n

O número a ser dividido.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

d

O número o qual n é dividido.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

round

Veja a função gmp_div_q() para uma descrição do argumento round.

Valor Retornado

Retorna um array, com o primeiro elemento sendo [n/d] (o resultado inteiro da divisão) e o segundo sendo (n - [n/d] * d) (o resto da divisão).

Exemplos

Exemplo #1 Divisão de números GMP

<?php
$a = gmp_init("0x41682179fbf5");
$res = gmp_div_qr($a, "0xDEFE75");
printf("Result is: q - %s, r - %s",
     gmp_strval ($res[0]), gmp_strval ($res[1]));
?>
