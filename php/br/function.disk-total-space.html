disk_total_space

(PHP 4 >= 4.1.0, PHP 5)

disk_total_space — Retorna o tamanho total do diretório

Descrição

float disk_total_space ( string $directory )

Dado uma string contendo um diretório, esta função retornará o numero total de bytes do sistema de arquivos ou a partição de disco.

Parâmetros

directory

Um diretório do sistema de arquivo ou partição de disco.

Valor Retornado

Retorna o número total de bytes como um float.

Exemplos

Exemplo #1 Exemplo da disk_total_space()

<?php
// $df contém o numero total de bytes disponíveis em "/"
$df = disk_total_space("/");

// On Windows:
disk_total_space("C:");
disk_total_space("D:");
?>

Notas

Nota: Esta função não trabalha com arquivos remotos, de forma que o arquivo a ser examinado precisa ser acessível pelo sistema de arquivos do servidor.
