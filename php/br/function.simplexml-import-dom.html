simplexml_import_dom

(PHP 5)

simplexml_import_dom — Recebe um objeto SimpleXMLElement de um nó DOM.

Descrição

SimpleXMLElement simplexml_import_dom ( DOMNode $node [, string $class_name ] )

Esta função recebe um nó de um documento DOM e o transforma em um nó SimpleXML. O novo objeto pode ser utilizado como um elemento SimpleXML nativo.

Parâmetros

node

Um nó DOM

class_name

Você pode utilizar o parâmetro opcional simplexml_import_dom() para retornar um objeto da classe especificada. Esta classe deve extender a classe SimpleXMLElement.

Valor Retornado

Retorna um SimpleXMLElement ou FALSE em caso de falha.

Exemplos

Exemplo #1 Importing DOM

<?php
$dom = new DOMDocument;
$dom->loadXML('<books><book><title>blah</title></book></books>');
if (!$dom) {
    echo 'Error while parsing the document';
    exit;
}

$s = simplexml_import_dom($dom);

echo $s->book[0]->title; // blah
?>
