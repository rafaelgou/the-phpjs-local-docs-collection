printer_open

(PECL printer SVN)

printer_open — Abre uma conexão com a impressora

Descrição

mixed printer_open ([ string $devicename ] )

Esta função tenta abrir uma conexão com a impressora devicename, e retorna um manipulador em caso de sucesso ou FALSE em caso de falha.

Se não for dado um parâmetro, ele tenta abrir uma conexão com a impressora padrão (se não especificado no php.ini como printer.default_printer, o PHP tenta detecta-lo).

printer_open() também inicia um device context.

Exemplo #1 Exemplo printer_open()

<?php
$handle = printer_open("HP Deskjet 930c");
$handle = printer_open();
?>
