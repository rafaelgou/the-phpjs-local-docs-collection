ftp_close

(PHP 4 >= 4.2.0, PHP 5)

ftp_close — Fecha uma conexão FTP

Descrição

bool ftp_close ( resource $ftp_stream )

ftp_close() fecha o identificador de recurso dado e libera o resource.

Nota:

Após chamar esta função, você não pode mais usar a conexão FTP e deve criar uma nova com ftp_connect().

Parâmetros

ftp_stream

O identificador da conexão FTP.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplos

Exemplo #1 Exemplo ftp_close()

<?php

// define a conexão básica
$conn_id = ftp_connect($ftp_server);

// login com nome de usuário e senha
$login_result = ftp_login($conn_id, $ftp_user_name, $ftp_user_pass);

// mostra o diretório atual
echo ftp_pwd($conn_id); // /

// fecha esta conexão
ftp_close($conn_id);
?>
