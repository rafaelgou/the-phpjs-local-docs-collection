ibase_num_params

(PHP 5)

ibase_num_params — Retorna o número de parâmetros em uma consulta preparada

Descrição

int ibase_num_params ( resource $query )

Esta função retorna o número de parâmetros em uma consulta preparada especificada query. Este é o numero de argumentos que devem estar presentes quando usar a função ibase_execute().

Parâmetros

query

O manipulador da query preparada.

Valor Retornado

Retorna o número de parâmetros como um inteiro.

Veja Também

  • ibase_prepare() - Prepara uma query para execução posterior
  • ibase_param_info() - Retorna informação sobre um parâmetro em uma consulta preparada
