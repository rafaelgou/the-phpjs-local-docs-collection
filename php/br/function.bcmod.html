bcmod

(PHP 4, PHP 5)

bcmod — Obtém o módulo de um número com precisão arbitrária

Descrição

string bcmod ( string $left_operand , string $modulus )

Obter o módulo do left_operand usando modulus.

Parâmetros

left_operand

O operando da esquerda, como uma string.

modulus

O módulo, como uma string.

Valor Retornado

Retorna o módulo como uma string, ou NULL se modulus é 0.

Exemplos

Exemplo #1 Exemplo da bcmod()

<?php
echo bcmod('4', '2'); // 0
echo bcmod('2', '4'); // 2
?>
