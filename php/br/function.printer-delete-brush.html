printer_delete_brush

(PECL printer SVN)

printer_delete_brush — Exclui um pincel

Descrição

bool printer_delete_brush ( resource $handle )

Esta função exclui o pincel selecionado. Para um exemplo veja printer_select_brush(). Retorna TRUE em caso de sucesso ou FALSE em caso de falha. handle deve ser um manipulador válido para um pincel.
