ibase_commit_ret

(PHP 5)

ibase_commit_ret — Grava uma transação sem fechá-la

Descrição

bool ibase_commit_ret ([ resource $link_or_trans_identifier ] )

Grava uma transação sem fechá-la.

Parâmetros

link_or_trans_identifier

Se for utilizada sem um argumento, esta função grava a transação padrão da conexão padrão. Se o argumento é um identificador de conexão, a transação padrão da conexão correspondente será gravada. Se o argumento for um identificador de transação, a transação correspondente será salva. O contexto da transação será retido, então comandos executados a partir desta transação não seram invalidados.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
