session_unset

(PHP 4, PHP 5)

session_unset — Libera todas as variáveis de sessão

Descrição

void session_unset ( void )

A função session_unset() libera todas as variáveis da sessão atualmente registradas.

Valor Retornado

Não há valor retornado.

Notas

Nota:

Se $_SESSION (ou $HTTP_SESSION_VARS para PHP 4.0.6 ou inferior) é usada, utilize unset() para desregistrar a variável da sessão. i.e. unset($_SESSION['varname']);.

Cuidado

NÃO elimina completamente $_SESSION com unset($_SESSION) enquanto esta desabilitará o registro de variáveis de sessão pela $_SESSION superglobal.
