sqrt

(PHP 4, PHP 5)

sqrt — Raiz quadrada

Descrição

float sqrt ( float $arg )

Retorna a raiz quadrada de arg.

Parâmetros

arg

O argumento para processar

Valor Retornado

A raiz quadrada de arg ou o valor especial NAN para números negativos.

Exemplos

Exemplo #1 Exemplo da sqrt()

<?php
// Precisão depende de sua diretiva precision
echo sqrt(9); // 3
echo sqrt(10); // 3.16227766 ...
?>
