imagefilledrectangle

(PHP 4, PHP 5)

imagefilledrectangle — Desenha um retângulo prenchido

Descrição

int imagefilledrectangle ( resource $image , int $x1 , int $y1 , int $x2 , int $y2 , int $color )

imagefilledrectangle() cria um retângulo prenchido da cor color na imagem image começando nas coordenadas de cima aesquerda x1, y1 e terminando nas coordenadas de baixo a direita x2, y2. 0, 0 é o canto superior esquerdo da imagem.
