gmp_divexact

(PHP 4 >= 4.0.4, PHP 5)

gmp_divexact — Divisão exata de números

Descrição

resource gmp_divexact ( resource $n , resource $d )

Divide n por d, usando um algaritimo rápido "exact division". Esta função produz resultados corretos apenas quando sabe anteriormente que d divide n.

Parâmetros

n

O número a ser dividido.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

d

O número o qual a será dividido.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Um resource de um número GMP.

Exemplos

Exemplo #1 Exemplo gmp_divexact()

<?php
$div1 = gmp_divexact("10", "2");
echo gmp_strval($div1) . "\n";

$div2 = gmp_divexact("10", "3"); // resultado errado
echo gmp_strval($div2) . "\n";
?>
