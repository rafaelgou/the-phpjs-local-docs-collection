printer_draw_line

(PECL printer SVN)

printer_draw_line — Desenha uma linha

Descrição

void printer_draw_line ( resource $printer_handle , int $from_x , int $from_y , int $to_x , int $to_y )

A função simplesmente desenha uma linha da posição from_x, from_y para a posição to_x, to_y usando a caneta selecionada. printer_handle deve ser um manipulador válido para uma impressora.

Exemplo #1 Exemplo printer_draw_line()

<?php
$handle = printer_open();
printer_start_doc($handle, "My Document");
printer_start_page($handle);

$pen = printer_create_pen(PRINTER_PEN_SOLID, 30, "000000");
printer_select_pen($handle, $pen);

printer_draw_line($handle, 1, 10, 1000, 10);
printer_draw_line($handle, 1, 60, 500, 60);

printer_delete_pen($pen);

printer_end_page($handle);
printer_end_doc($handle);
printer_close($handle);
?>
