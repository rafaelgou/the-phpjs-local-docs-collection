parsekit_compile_string

(PECL parsekit >= 0.2.0)

parsekit_compile_string — Compila uma string de código PHP e retorna um array com o resultado

Descrição

array parsekit_compile_string ( string $phpcode [, array &$errors [, int $options ]] )
Aviso

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome, incluindo toda documentação pode ser modificado sem aviso em futuras versões do PHP. Esta função deve ser usada por sua própria conta e risco.

Parâmetros

phpcode

Uma string contendo phpcode. Similar ao argumento para eval().

errors

Um hash 2D de erros (incluindo erros fatais) encontrados durante a compilação. Retornado por referência.

options

Uma das constantes PARSEKIT_QUIET ou PARSEKIT_SIMPLE. Para produzir vários tipos de avisos na saída retornada.

Valor Retornado

Retorna uma complexa estrutura de array multi-camada como visto abaixo.

Exemplos

Exemplo #1 Exemplo da parsekit_compile_string()

<?php
  $ops = parsekit_compile_string('
echo "Foo\n";
', $errors, PARSEKIT_QUIET);

  var_dump($ops);
?>
