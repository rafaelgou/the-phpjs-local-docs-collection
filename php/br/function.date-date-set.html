date_date_set

(PHP 5 >= 5.2.0)

date_date_set — Define a data

Descrição

void date_date_set ( DateTime $object , int $year , int $month , int $day )
void DateTime::setDate ( int $year , int $month , int $day )

Parâmetros

object

Objeto DateTime.

year

Ano da data.

month

Mês da data.

day

Dia da data.

Valor Retornado

Retorna NULL em caso de sucesso ou FALSE em falha.
