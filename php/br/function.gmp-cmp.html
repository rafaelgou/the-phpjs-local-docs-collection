gmp_cmp

(PHP 4 >= 4.0.4, PHP 5)

gmp_cmp — Compara números

Descrição

int gmp_cmp ( resource $a , resource $b )

Compara dois números.

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

b

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Retorna um valor positivo se a > b, zero se a = b e um valor negativo se a < b.

Exemplos

Exemplo #1 Exemplo da gmp_cmp()

<?php
$cmp1 = gmp_cmp("1234", "1000"); // maior do que
$cmp2 = gmp_cmp("1000", "1234"); // menor do que
$cmp3 = gmp_cmp("1234", "1234"); // igual a 
       
echo "$cmp1 $cmp2 $cmp3\n";
?>

O exemplo acima irá imprimir:

1 -1 0
