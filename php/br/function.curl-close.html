curl_close

(PHP 4 >= 4.0.2, PHP 5)

curl_close — Fecha uma sessão cURL

Descrição

void curl_close ( resource $ch )

Fecha uma sessão cURL e libera todos os recursos. O controlador cURL, ch, também é deletado.

Parâmetros

ch

Um manipulador cURL retornado por curl_init().

Valor Retornado

Não há valor retornado.

Exemplos

Exemplo #1 Iniciando uma nova sessão cURL e acessando uma página web

<?php
// Cria uma nova sessão cURL
$ch = curl_init();

// Configura uma URL e outras opções apropriadas
curl_setopt($ch, CURLOPT_URL, "http://www.exemplo.com.br/");
curl_setopt($ch, CURLOPT_HEADER, 0);

// Executa a sessão
curl_exec($ch);

// Fecha a sessão cURL e libera recursos
curl_close($ch);
?>
