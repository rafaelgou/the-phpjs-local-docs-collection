pg_port

(PHP 4, PHP 5)

pg_port — Retorna o número da porta associada com a conexão.

Descrição

int pg_port ( resource $connection )

pg_port() retorna o número da porta a qual o recurso de conexão PostgreSQL connection está conectado.
