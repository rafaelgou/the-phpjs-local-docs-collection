mysql_field_flags

(PHP 4, PHP 5)

mysql_field_flags — Obtém as flags associadas ao campo especificado em um resultado

Descrição

string mysql_field_flags ( resource $result , int $field_offset )

mysql_field_flags() retorna as flags do campo especificado. As flags são mostradas como uma única palavra por flag separada por um único espaço, assim você pode dividir o valor retornado usando explode().

Parâmetros

result

The result resource that is being evaluated. This result comes from a call to mysql_query().

field_offset

The numerical field offset. The field_offset starts at 0. If field_offset does not exist, an error of level E_WARNING is also issued.

Valor Retornado

Retorna uma string com as flags associadas ao resultado, ou false em caso de falha.

As seguintes flags são reportadas, se a sua versão do MySQL for atual o suficiente para suporta-las: "not_null", "primary_key", "unique_key", "multiple_key", "blob", "unsigned", "zerofill", "binary", "enum", "auto_increment" e "timestamp".

Exemplos

Exemplo #1 Um exemplo de mysql_field_flags()

<?php
$result = mysql_query("SELECT id,email FROM people WHERE id = '42'");
if (!$result) {
    echo 'Não foi possível executar a consulta: ' . mysql_error();
    exit;
}
$flags = mysql_field_flags($result, 0);

echo $flags;
print_r(explode(' ', $flags));
?>

O exemplo acima irá imprimir algo similar à:

not_null primary_key auto_increment
Array
(
    [0] => not_null
    [1] => primary_key
    [2] => auto_increment
)

Notas

Nota:

Para compatibilidade com versões anteriores, o seguinte apelido obsoleto pode ser usado: mysql_fieldflags()
