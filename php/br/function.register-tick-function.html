register_tick_function

(PHP 4 >= 4.0.3, PHP 5)

register_tick_function — Registra uma função para executar a cada instante

Descrição

bool register_tick_function ( callback $function [, mixed $arg [, mixed $... ]] )

Registra a dada function para ser executada quando um tick é chamado.

Parâmetros

function

O nome da função como string, ou um array consistindo de um objeto e um método.

arg

...

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplos

Exemplo #1 Exemplo de uso da register_tick_function()

<?php
// using a function as the callback
register_tick_function('my_function', true);

// using an object->method
$object = new my_class();
register_tick_function(array(&$object, 'my_method'), true);
?>

Notas

Aviso

register_tick_function() não deve ser usado com threaded web server modules. Ticks não funcionam em modo ZTS e pode travar seu web server.
