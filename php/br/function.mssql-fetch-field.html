mssql_fetch_field

(PHP 4, PHP 5, PECL odbtp >= 1.1.1)

mssql_fetch_field — Retorna informação sobre o campo

Descrição

object mssql_fetch_field ( resource $result [, int $field_offset ] )

mssql_fetch_field() pode ser usado para obter informação sobre campos de um certo resultado de query.

Parâmetros

result

O resource do resultado que está sendo avaliado. Este resultado vem de uma chamada a mssql_query().

field_offset

O índice numérico do campo. Se o índice do campo não é especificado, o próximo campo que não foi recebido por esta função é recuperado. O field_offset inicia em 0.

Valor Retornado

Retorna um objeto contendo informação do campo.

As propriedades do objeto são:

  • name - nome da coluna. Se a coluna é resultado de alguma função, esta propriedade é estabelecida como computed#N, aonde #N é um número de série.
  • column_source - a tabela de origem da coluna.
  • max_length - o limite de tamanho da coluna.
  • numeric - 1 se a coluna é numérica.
  • type - O tipo da coluna.
