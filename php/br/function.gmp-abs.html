gmp_abs

(PHP 4 >= 4.0.4, PHP 5)

gmp_abs — Valor absoluto

Descrição

resource gmp_abs ( resource $a )

Obtém o valor absoluto de um número.

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Retorna o valor absoluto de a, como um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_abs()

<?php
$abs1 = gmp_abs("274982683358");
$abs2 = gmp_abs("-274982683358");
       
echo gmp_strval($abs1) . "\n";
echo gmp_strval($abs2) . "\n";
?>

O exemplo acima irá imprimir:

274982683358
274982683358
