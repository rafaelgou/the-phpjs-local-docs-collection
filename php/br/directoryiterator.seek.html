DirectoryIterator::seek

(PHP 5 >= 5.3.0)

DirectoryIterator::seek — Procura uma determinada posição no item do DirectoryIterator

Descrição

public void DirectoryIterator::seek ( int $posicao )

Procura uma determinada posição no item do DirectoryIterator.

Parâmetros

posicao

Posição de base zero para procurar.

Valor Retornado

Não há valor retornado.

Exemplos

Exemplo #1 Exemplo do DirectoryIterator::seek()

Procurar o quarto item no diretório que contém o script. Os dois primeiros geralmente são . e ..

<?php
$iterator = new DirectoryIterator(dirname(__FILE__));
$iterator->seek(3);
if ($iterator->valid()) {
    echo $iterator->getFilename();
} else {
    echo 'Sem arquivos na posição 3';
}
?>
