win32_ps_stat_proc

(PECL win32ps >= 1.0.1)

win32_ps_stat_proc — Estatísticas de um processo

Descrição

array win32_ps_stat_proc ([ int $pid ] )

Obtem estatísticas sobre o processo com ID pid.

Parâmetros

pid

O ID do processo a ser investigado. Se omitido, o ID do processo atual é usado.

Valor Retornado

Retorna FALSE em caso de falha, ou um array com as seguintes informações:

pid

O ID do processo.

exe

O caminho para o executável.

mem

Um array contendo informação sobre indicadores de utilização de memória: page_fault_count, peak_working_set_size, working_set_size, quota_peak_paged_pool_usage, quota_paged_pool_usage, quota_peak_non_paged_pool_usage, quota_non_paged_pool_usage, pagefile_usage and peak_pagefile_usage.

tms

Um array contendo informação com inidicadores de utilização de tempo de CPU: created, kernel e user.
