ovrimos_connect

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_connect — Conecta com o banco de dados especificado

Descrição

int ovrimos_connect ( string $host , string $db , string $user , string $password )

ovrimos_connect() é usado para conectar com o banco de dados especificado.

ovrimos_connect() retorna um id de conexão (maior do que 0) ou 0 em caso de falha. O funcionamento de host e db são aqueles usados em todos os lugares da API do Ovrimos. host é o nome de um servidor ou o endereço IP e db é ou o nome do banco de dados, ou uma string contendo o número da porta.

Exemplo #1 Exemplo ovrimos_connect()

<?php
$conn = ovrimos_connect("server.domain.com", "8001", "admin", "password");
if ($conn != 0) {
    echo "Connection ok!";
    $res=ovrimos_exec($conn, "select table_id, table_name from sys.tables");
    if ($res != 0) {
        echo "Statement ok!";
        ovrimos_result_all($res);
        ovrimos_free_result($res);
    }
    ovrimos_close($conn);
}
?>
O exemplo acima irá conectar com o banco de dados e mostrar a tabela especificada.
