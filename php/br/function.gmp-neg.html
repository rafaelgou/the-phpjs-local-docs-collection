gmp_neg

(PHP 4 >= 4.0.4, PHP 5)

gmp_neg — Nega o número

Descrição

resource gmp_neg ( resource $a )

Retorna o valor negativo de um número.

Parâmetros

a

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

Valor Retornado

Retorna -a, como um número GMP.

Exemplos

Exemplo #1 Exemplo da gmp_neg()

<?php
$neg1 = gmp_neg("1");
echo gmp_strval($neg1) . "\n";
$neg2 = gmp_neg("-1");
echo gmp_strval($neg2) . "\n";
?>

O exemplo acima irá imprimir:

-1
1
