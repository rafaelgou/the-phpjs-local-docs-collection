sybase_select_db

(PHP 4, PHP 5)

sybase_select_db — Seleciona um banco de dados Sybase

Descrição

bool sybase_select_db ( string $database_name [, resource $link_identifier ] )

sybase_select_db() define o banco de dados ativo que é associado ao identificador de conexão especificado. Se não for especificado um identificador de conexão, a última conexão aberta é assumida. Se não houver conexão aberta, a função tentará establecer uma conexão como se sybase_connect() fosse utilizada, e irá usa-la.

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Todas as próximas chamadas a sybase_query() serão feitas no banco de dados ativo.

Veja também sybase_connect(), sybase_pconnect() e sybase_query()
