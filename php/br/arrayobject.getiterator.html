ArrayObject::getIterator

(PHP 5 >= 5.0.0)

ArrayObject::getIterator — Retorna o iterador do ArrayObject

Descrição

ArrayIterator ArrayObject::getIterator ( void )

Cria um novo iterador a partir de uma instância de ArrayObject.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

Um iterador de um ArrayObject.

Exemplos

Exemplo #1 Exemplo de ArrayObject::getIterator()

<?php
$array = array('1' => 'um',
               '2' => 'dois',
               '3' => 'três');

$arrayobject = new ArrayObject($array);

$iterator = $arrayobject->getIterator();

while($iterator->valid()) {
    echo $iterator->key() . ' => ' . $iterator->current() . "\n";

    $iterator->next();
}
?>

O exemplo acima irá imprimir:

1 => um
2 => dois
3 => três
