cosh

(PHP 4 >= 4.1.0, PHP 5)

cosh — Cosseno hiperbólico

Descrição

float cosh ( float $arg )

Retorna o cosseno hiperbólico de arg, definido por (exp(arg) + exp(-arg))/2.

Parâmetros

arg

O argumento para processar

Valor Retornado

O cosseno hiperbólico de arg
