mssql_execute

(PHP 4 >= 4.0.7, PHP 5, PECL odbtp >= 1.1.1)

mssql_execute — Executa uma stored procedure num banco de dados de um servidor MS SQL

Descrição

mixed mssql_execute ( resource $stmt [, bool $skip_results ] )
Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Nota: Se o stored procedure retorna parâmetros ou retorna um valor, estes estarão disponiveis após a chamada a mssql_execute() a menos que o stored procedure retorne mais do que um conjunto de resultados. Neste caso use mssql_next_result() para mudar entre os resulatdo. Quando o ultimo resultado tiver sido processado os parâmetors de saída e os valores de retorno estarão disponivéis.

Veja também mssql_bind(), mssql_free_statement(), e mssql_init().
