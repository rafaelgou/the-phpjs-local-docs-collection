array_combine

(PHP 5)

array_combine — Cria um array usando um array para chaves e outro para valores

Descrição

array array_combine ( array $keys , array $values )

Cria um array usando os valores do array keys como chaves e os valores do array values como valores correspondentes.

Parâmetros

keys

Array a ser usado como chaves. Valores ilegais para chave serão convertidos para string.

values

Array a ser usado como valores

Valor Retornado

Retorna o array combinado, FALSE se o número de elementos de cada array não são iguais.

Erros

Dispara E_WARNING se o número de elementos em keys e values não correspondem.

Changelog

Versão Descrição
5.4.0 Versões anteriores disparavam E_WARNING e retornavam FALSE para arrays vazios.

Exemplos

Exemplo #1 Um simples exemplo usando array_combine()

<?php
$a = array('verde', 'vermelho', 'amarelo');
$b = array('abacate', 'maçã', 'banana');
$c = array_combine($a, $b);

print_r($c);
?>

O exemplo acima irá imprimir:

Array
(
    [green]  => abacate
    [red]    => maçã
    [yellow] => banana
)
