ArrayIterator::current

(PHP 5 >= 5.0.0)

ArrayIterator::current — Retorna o valor do elemento atual do array

Descrição

mixed ArrayIterator::current ( void )

Obtém o atual elemento do array

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

O atual elemento do array.

Exemplos

Exemplo #1 Exemplo de ArrayIterator::current()

<?php
$array = array('1' => 'um',
               '2' => 'dois',
               '3' => 'três');

$arrayobject = new ArrayObject($array);

for($iterator = $arrayobject->getIterator();
    $iterator->valid();
    $iterator->next()) {

    echo $iterator->key() . ' => ' . $iterator->current() . "\n";
}
?>

O exemplo acima irá imprimir:

1 => um
2 => dois
3 => três
