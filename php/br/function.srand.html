srand

(PHP 4, PHP 5)

srand — Semeia o gerador de números aletórios

Descrição

void srand ([ int $seed ] )

Semeia o gerador de números aleatórios com seed ou com um valor aleatório se seed não é dado.

Nota: A partir do PHP 4.2.0, não é mais necessário semear o gerador de números aleatórios com srand() ou mt_srand() pois isso é feito automaticamente.

Parâmetros

seed

Valor seed opcional

Changelog

Versão Descrição
Since 4.2.0 O seed tornou-se opcional e o padrão é um valor aleatório se omitido.

Exemplos

Exemplo #1 Exemplo da srand()

<?php
// semeira com microsegundos
function make_seed() {
    list($usec, $sec) = explode(' ', microtime());
    return (float) $sec + ((float) $usec * 100000);
}
srand(make_seed());
$randval = rand();
?>

Veja Também

  • rand() - Gera um inteiro aleatório
  • getrandmax() - Retorna o maior valor aletório possível
  • mt_srand() - Semeia o gerador melhorado de números aleatórios
