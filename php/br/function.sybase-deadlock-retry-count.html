sybase_deadlock_retry_count

(PHP 4 >= 4.3.0, PHP 5)

sybase_deadlock_retry_count — Define o número de tentativas em caso de travamento

Descrição

void sybase_deadlock_retry_count ( int $retry_count )

Usando sybase_deadlock_retry_count(), o número de tentativas pode ser definido em caso de travamentos. Por padrão, cada travamento é tentado novamente um número infinito de vezes ou até que o processo seja morto pelo Sybase, o script em execução é morto (por exemplo, por set_time_limit()) ou a consulta seja bem sucedida.

Parâmetros

retry_count

Valores para retry_count
-1 Tentar novamente para sempre (padrão)
0 Não tentar novamente
n Tentar novamente n vezes

Valor Retornado

Não há valor retornado.

Notas

Nota: Esta função somente está disponível quando utilizando a biblioteca de interface CT do Sybase, mas não pela biblioteca DB.
