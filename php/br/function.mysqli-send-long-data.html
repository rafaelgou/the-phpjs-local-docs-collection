mysqli_send_long_data — Alias for mysqli_stmt_send_long_data()

Descrição

This function is an alias of mysqli_stmt_send_long_data().

Aviso

Esta função tornou-se OBSOLETA desde o PHP 5.3.0 e foi REMOVIDA desde o PHP 5.4.0.
