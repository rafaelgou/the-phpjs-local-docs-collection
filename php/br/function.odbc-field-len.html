odbc_field_len

(PHP 4, PHP 5)

odbc_field_len — Obtém o tamanho (precisão) de um campo

Descrição

int odbc_field_len ( resource $result_id , int $field_number )

Obtém o tamanho do campo referenciado pelo número no dado identificador de resultado.

Parâmetros

result_id

O identificador do resultado.

field_number

O número do campo. A numeração dos campos inicia em 0.

Valor Retornado

Retorna o nome do campo como uma string, ou FALSE em erro.
