shmop_delete

(PHP 4 >= 4.0.4, PHP 5)

shmop_delete — Deleta um bloco de memória compartilhada

Descrição

bool shmop_delete ( int $shmid )

shmop_delete() é usada para deletar um bloco de memória compartilhada.

Parâmetros

shmid

O identificaor de bloco de memória compartilhada criado por shmop_open().

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplos

Exemplo #1 Deletando um bloco de memória compartilhada

<?php
shmop_delete($shm_id);
?>

Esse exemplo irá deletar um bloco de memória compartilhada identificado por $shm_id.
