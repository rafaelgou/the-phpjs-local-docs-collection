touch

(PHP 4, PHP 5)

touch — Muda o tempo de modificação do arquivo

Descrição

bool touch ( string $filename [, int $time [, int $atime ]] )

Tenta mudar o tempo de acesso e modificação do arquivo filename para o valor informado em time. Note que a data/hora do último acesso do arquivo sempre é alterado, independentemente do número de parâmetros.

Se o arquivo não existe, ele será criado.

Parâmetros

filename

O nome do arquivo a ser atualizado.

time

Hora da atualização. Se time não é fornecido, o tempo atual do sistema é usado.

atime

Se informado, o tempo do acesso do dado arquivo é definido para o valor de atime

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplos

Exemplo #1 Exemplo da função touch()

<?php
if (touch ($nomedoarquivo)) {
    print "o tempo do $nomedoarquivo foi modificado para o dia e hora atual";
} else {
    print "Desculpe, não foi possivel modificar o tempo de $nomedoarquivo";
}
?>

Notas

Aviso

Não é atualmente possível modificar a hora de modificação de um diretório com esta função no Windows.
