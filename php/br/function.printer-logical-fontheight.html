printer_logical_fontheight

(PECL printer SVN)

printer_logical_fontheight — Obtém a altura lógica da fonte

Descrição

int printer_logical_fontheight ( resource $handle , int $height )

A função calcula a altura logica da fonte de height. handle deve ser um manipulador válido para a impressora.

Exemplo #1 Exemplo printer_logical_fontheight()

<?php
$handle = printer_open();
echo printer_logical_fontheight($handle, 72);
printer_close($handle);
?>
