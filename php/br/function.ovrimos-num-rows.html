ovrimos_num_rows

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_num_rows — Retorna o número de linhas afetadas por uma operação update

Descrição

int ovrimos_num_rows ( int $result_id )

Obtém o número de linhas afetadas por uma operação de atualização.

Parâmetros

result_id

Um identificador de resultado, retornado pela ovrimos_execute() ou ovrimos_exec().

Valor Retornado

Retorna o número de linhas como um inteiro, ou FALSE em erro.
