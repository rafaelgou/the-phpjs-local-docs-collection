deg2rad

(PHP 4, PHP 5)

deg2rad — Converte o número em graus ao equivalente em radianos

Descrição

float deg2rad ( float $number )

Esta função converte number de graus ao equivalente em radianos.

Parâmetros

number

Valor do ângulo em graus

Valor Retornado

O radiano equivalente ao number

Exemplos

Exemplo #1 Exemplo da deg2rad()

<?php

echo deg2rad(45); // 0.785398163397
var_dump(deg2rad(45) === M_PI_4); // bool(true)

?>
