strpbrk

(PHP 5)

strpbrk — Procura na string por um dos caracteres de um conjunto

Descrição

string strpbrk ( string $haystack , string $char_list )

strpbrk() busca na string haystack por uma char_list.

Parâmetros

haystack

A string onde char_list irá verificar.

char_list

Este parâmetro é case sensitive.

Valor Retornado

Retorna uma string iniciando do caractere encontrado, ou FALSE se ele não é encontrado.

Exemplos

Exemplo #1 Exemplo da strpbrk()

<?php

$text = 'This is a Simple text.';

// this echoes "is is a Simple text." because 'i' is matched first
echo strpbrk($text, 'mi');

// this echoes "Simple text." because chars are case sensitive
echo strpbrk($text, 'S');
?>
