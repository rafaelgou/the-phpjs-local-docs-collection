call_user_method_array

(PHP 4 >= 4.0.5, PHP 5)

call_user_method_array — Chama métodos de usuário dado uma matriz de parâmetros [obsoleto]

Descrição

mixed call_user_method_array ( string $method_name , object &$obj , array $params )
Aviso

A função call_user_method_array() é obsoleta a partir do PHP 4.1.0.

Exemplos

Exemplo #1 Alternativo call_user_method_array()

<?php
call_user_func_array(array($obj, $method_name), $params);
call_user_func_array(array(&$obj, $method_name), $params); // PHP 4
?>

Veja Também

  • call_user_func_array() - Chama uma dada função de usuário com um array de parâmetros
  • call_user_func() - Chama uma função de usuário dada pelo primeiro parâmetro
