odbc_field_scale

(PHP 4, PHP 5)

odbc_field_scale — Obtém a precisão de um campo

Descrição

int odbc_field_scale ( resource $result_id , int $field_number )

Obtém a precisão do campo referenciado pelo número em um dado identificador de resultado.

Parâmetros

result_id

O identificador de resultado.

field_number

O número do campo. A numeração do campo inicia em 1.

Valor Retornado

Retorna a precisão do campo como um inteiro, ou FALSE em erro.
