get_headers

(PHP 5)

get_headers — Retorna todos os cabeçalhos enviados pelo servidor em resposta à requisição HTTP

Descrição

array get_headers ( string $url [, int $format ] )

get_headers() retorna um array com os cabeçalhos enviados pelo servidor em resposta à requisição HTTP.

Parâmetros

url

A URL alvo.

format

Se o parâmetro opcional format tiver valor 1, get_headers() avalia a resposta e configura as chaves do array.

Valor Retornado

Retorna um array indexado ou associativo com os cabeçalhos, ou FALSE em caso de falha.

Changelog

Versão Descrição
5.1.3 Essa função agora usa o contexto de streaming padrão, que pode ser alterado com a função stream_context_get_default().

Exemplos

Exemplo #1 Exemplo de uso de get_headers()

<?php
$url = 'http://www.example.com';

print_r(get_headers($url));

print_r(get_headers($url, 1));
?>
