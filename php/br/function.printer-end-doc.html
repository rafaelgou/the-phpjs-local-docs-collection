printer_end_doc

(PECL printer SVN)

printer_end_doc — Fecha o documento

Descrição

bool printer_end_doc ( resource $handle )

Fecha um novo documento na fila de impressão. O documento agora estará pronto para ser impresso. Para um exemplo veja printer_start_doc(). handle deve ser um manipulador válido para a impressora.
