odbc_procedurecolumns

(PHP 4, PHP 5)

odbc_procedurecolumns — Obtém informações sobre parâmetros para procedures

Descrição

resource odbc_procedurecolumns ( resource $connection_id [, string $qualifier ], string $owner , string $proc , string $column )

Retorna uma lista de parâmetros de entrada e saída, assim como as colunas que fazem parte do conjunto de resultados para os procedures especificados. Retorna um idenficador de resultado ODBC ou FALSE em caso de falha.

O conjunto de resultados tem as seguintes colunas:

  • PROCEDURE_QUALIFIER
  • PROCEDURE_OWNER
  • PROCEDURE_NAME
  • COLUMN_NAME
  • COLUMN_TYPE
  • DATA_TYPE
  • TYPE_NAME
  • PRECISION
  • LENGTH
  • SCALE
  • RADIX
  • NULLABLE
  • REMARKS

O conjunto de resultados é ordenado por PROCEDURE_QUALIFIER, PROCEDURE_OWNER, PROCEDURE_NAME e COLUMN_TYPE.

Os argumentos owner, proc and column aceitam modelos de busca ('%' para buscar zero ou mais caracteres e '_' para buscar um único caractere).
