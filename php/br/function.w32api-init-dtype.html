w32api_init_dtype

(PHP 4 >= 4.2.0)

w32api_init_dtype — Cria uma instância do tipo de dado typename e atribui ao mesmo os valores passados

Descrição

resource w32api_init_dtype ( string $typename , mixed $value [, mixed $... ] )

Essa função cria uma instância do tipo de dado com nome typename, preenchendo-a com os valores do tipo de dado.

Parâmetros

typename

O parâmetro typename é sensível ao caso (distingue minúsculas de maiúsculas).

value

Você deve dar os valores na mesma ordem que você definiu o tipo de dado com a função w32api_deftype().

...

Valor Retornado

Retorna um recurso (resource) dynaparm.

Notas

Aviso

Esta função é EXPERIMENTAL. O comportamento desta função, seu nome, incluindo toda documentação pode ser modificado sem aviso em futuras versões do PHP. Esta função deve ser usada por sua própria conta e risco.
