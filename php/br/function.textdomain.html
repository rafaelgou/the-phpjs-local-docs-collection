textdomain

(PHP 4, PHP 5)

textdomain — Seta o domínio padrão

Descrição

string textdomain ( string $text_domain )

Esta função permite à você confiugrar o domínio padrão pelo qual as chamadas são feitas usando a função gettext(), geralmente o nomeado após a aplicação. O domínio padrão é retornado. Chame ela com o parâmetro NULL para pegar as configurações atuais sem alterar isto.
