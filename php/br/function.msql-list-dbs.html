msql_list_dbs

(PHP 4, PHP 5)

msql_list_dbs — Lista os banco de dados mSQL no servidor

Descrição

resource msql_list_dbs ([ resource $link_identifier ] )

msql_list_dbs() listará os banco de dados disponíveis no especificado link_identifier.

Parâmetros

link_identifier

A conexão mSQL. Se não for especificado, a ultima conexão aberta por msql_connect() é assumida. Se uma conexão assim não for encontrada, a função irá tentar establecer uma conexão como se a função msql_connect() fosse chamada, e irá usa-la.

Valor Retornado

Retorna um conjunto de resultado que pode ser percorrido com alguma função que obtém os conjuntos de resultados, como a msql_fetch_array(). Em falha esta função retornará FALSE.
