vprintf

(PHP 4 >= 4.1.0, PHP 5)

vprintf — Mostra uma string formatada

Descrição

int vprintf ( string $formato , array $args )

Mostra uma string formatada de acordo com o formato (o qual é descrito na documentação para a função sprintf()).

Funciona como printf() mas aceita uma matriz de argumentos, ao invés de um número variavel de argumentos.

Parâmetros

format

Veja sprintf() para uma descrição de format.

args

Valor Retornado

Retorna o tamanho da string mostrada.

Veja Também

  • sprintf() - Retorna a string formatada
  • vsprintf() - Retorna uma string formatada
