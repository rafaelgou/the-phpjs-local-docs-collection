is_numeric

(PHP 4, PHP 5)

is_numeric — Informa se a variável é um número ou uma string numérica

Descrição

bool is_numeric ( mixed $var )

Verifica se a dada variável é numérica. Strings numéricas consistem de um sinal opcional, algum número de dígitos, opcional parte decimal e opcional parte exponencial. Assim +0123.45e6 é um válido valor numérico. Notação hexadecimal (0xFF) é permitida também mas somente sem sinal, decimal e parte exponencial.

Parâmetros

var

A variável a ser avaliada.

Valor Retornado

Retorna TRUE se var é um número ou uma string numérica, FALSE caso contrário.
