preg_last_error

(PHP 5 >= 5.2.0)

preg_last_error — Retorna o código de erro da última regex PCRE executada

Descrição

int preg_last_error ( void )

Retorna o código de erro da última regex PCRE executada.

Exemplo #1 Exemplo da preg_last_error()

<?php

preg_match('/(?:\D+|<\d+>)*[!?]/', 'foobar foobar foobar');

if (preg_last_error() == PREG_BACKTRACK_LIMIT_ERROR) {
    print 'Backtrack limit was exhausted!';
}

?>

O exemplo acima irá imprimir:

Backtrack limit was exhausted!

Valor Retornado

Retorna uma das seguintes constantes (explicadas nessa outra página):

  • PREG_NO_ERROR
  • PREG_INTERNAL_ERROR
  • PREG_BACKTRACK_LIMIT_ERROR (veja também pcre.backtrack_limit)
  • PREG_RECURSION_LIMIT_ERROR (veja também pcre.recursion_limit)
  • PREG_BAD_UTF8_ERROR
  • PREG_BAD_UTF8_OFFSET_ERROR (desde o PHP 5.3.0)
