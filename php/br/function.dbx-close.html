dbx_close

(PHP 4 >= 4.0.6, PHP 5 <= 5.0.5, PECL dbx >= 1.1.0)

dbx_close — Encera uma conexão/banco de dados

Descrição

int dbx_close ( object $link_identifier )

Parâmetros

link_identifier

O link do objeto DBX para fechar.

Valor Retornado

Retorna 1 em successo e 0 em erros.

Exemplos

Exemplo #1 dbx_close() exemplo

<?php
$link = dbx_connect(DBX_MYSQL, "localhost", "db", "nome de usuário", "senha")
    or die ("Não foi possível conectar");

echo "Conexão realizada com sucesso";
dbx_close($link);
?>
