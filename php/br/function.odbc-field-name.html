odbc_field_name

(PHP 4, PHP 5)

odbc_field_name — Obtém o nome da coluna

Descrição

string odbc_field_name ( resource $result_id , int $field_number )

Obtém o nome do campo ocupando o número da coluna em um dado identificador de resultado.

Parâmetros

result_id

O identificador de resultado.

field_number

O número do campo. A numeração de campos iniciam em 1.

Valor Retornado

Retorna o nome do campo como uma string, ou FALSE em erro.
