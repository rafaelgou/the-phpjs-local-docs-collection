ovrimos_field_num

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_field_num — Retorna o índice da coluna (baseado em 1)

Descrição

int ovrimos_field_num ( int $result_id , string $field_name )

ovrimos_field_num() retorna o índice (baseado em 1) da coluna especificada por field_name, ou FALSE.
