ArrayIterator::key

(PHP 5 >= 5.0.0)

ArrayIterator::key — Retorna a chave do elemento atual do array

Descrição

mixed ArrayIterator::key ( void )

Esta função retorna a chave do elemento atual do array

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

A chave do atual elemento do array.

Exemplos

Exemplo #1 Exemplo de ArrayIterator::key()

<?php
$array = array('key' => 'value');

$arrayobject = new ArrayObject($array);
$iterator = $arrayobject->getIterator();

echo $iterator->key(); //key
?>
