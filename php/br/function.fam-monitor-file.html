fam_monitor_file

(PHP 5 <= 5.0.5)

fam_monitor_file — Monitora alterações em um arquivo

Descrição

resource fam_monitor_file ( resource $fam , string $filename )

Requisita monitoramento para um único arquivo. Um evento FAM irá ser gerado quando o status do arquivo (ex. o resultado da função stat() no arquivo) for alterado.

Os possíveis códigos do evento são descritos em detalhes na parte de constantes dessa seção.

Parâmetros

fam

Um resource representando a conexão para o serviço FAM retornado pela fam_open()

filename

Caminho para o arquivo a ser monitorado

Valor Retornado

Retorna um resource de monitoramento ou FALSE em erros.

Veja Também

  • fam_monitor_directory() - Monitora alterações em diretórios
  • fam_monitor_collection() - Monitora alterações de uma coleção de arquivos em um diretório
  • fam_cancel_monitor() - Finalizando monitoramento
  • fam_suspend_monitor() - Suspende monitoramento temporariamente
  • fam_resume_monitor() - Recomeça um monitoramento suspendido
