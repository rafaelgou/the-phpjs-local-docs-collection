array_product

(PHP 5 >= 5.1.0)

array_product — Calcula o produto dos valores de um array

Descrição

number array_product ( array $array )

array_product() retorna o produto dos valores de um array.

Parâmetros

array

O array.

Valor Retornado

Retorna o produto com um inteiro ou float.

Exemplos

Exemplo #1 Exemplo da array_product()

<?php

$a = array(2, 4, 6, 8);
echo "product(a) = " . array_product($a) . "\n";

?>

O exemplo acima irá imprimir:

product(a) = 384
