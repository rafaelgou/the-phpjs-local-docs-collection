bbcode_destroy

(PECL bbcode >= 0.9.0)

bbcode_destroy — Fecha o resource BBCode_container

Descrição

bool bbcode_destroy ( resource $bbcode_container )

Esta função fecha um resource aberto por bbcode_create().

Parâmetros

bbcode

Resource BBCode_Container retornado por bbcode_create().

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
