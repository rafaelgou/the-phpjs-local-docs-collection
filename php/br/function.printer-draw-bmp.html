printer_draw_bmp

(PECL printer SVN)

printer_draw_bmp — Desenha um bitmap

Descrição

void printer_draw_bmp ( resource $handle , string $filename , int $x , int $y )

Esta função simplesmente desenha um bitmap filename na posição x, y. handle deve ser um manipulador válido para uma impressora.

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Exemplo #1 Exemplo printer_draw_bmp()

<?php
$handle = printer_open();
printer_start_doc($handle, "My Document");
printer_start_page($handle);

printer_draw_bmp($handle, "c:\\image.bmp", 1, 1);

printer_end_page($handle);
printer_end_doc($handle);
printer_close($handle);
?>
