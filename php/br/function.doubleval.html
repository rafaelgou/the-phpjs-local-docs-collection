(PHP 4, PHP 5)

doubleval — Sinônimo de floatval()

Descrição

Esta função é um apelido para: floatval().

Changelog

Versão Descrição
4.2.0 doubleval() tornou-se um apelido para floatval(). Antes disso, somente doubleval() existia.
