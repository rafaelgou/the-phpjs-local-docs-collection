get_declared_interfaces

(PHP 5)

get_declared_interfaces — Retorna um array com todas as interfaces declaradas

Descrição

array get_declared_interfaces ( void )

Retorna as interfaces declaradas.

Valor Retornado

Essa função retorna um array com os nomes de interfaces declaradas no script atual.

Exemplos

Exemplo #1 Exemplo de uso de get_declared_interfaces()

<?php
print_r(get_declared_interfaces());
?>

O exemplo acima irá imprimir algo similar à:

Array
(
    [0] => Traversable
    [1] => IteratorAggregate
    [2] => Iterator
    [3] => ArrayAccess
    [4] => reflector
    [5] => RecursiveIterator
    [6] => SeekableIterator
)
