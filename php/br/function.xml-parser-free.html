xml_parser_free

(PHP 4, PHP 5)

xml_parser_free — Free an XML parser

Descrição

bool xml_parser_free ( resource $parser )

Libera um dado parser XML.

Parâmetros

parser
Uma referência para o analisador XML (to free) esgotar .

Valor Retornado

Esta função retorna FALSE se parser (analisador) não se refere a um analisador válido, senão ele (frees) esgota o analisador e retorna TRUE.
