printer_select_font

(PECL printer SVN)

printer_select_font — Seleciona uma fonte

Descrição

void printer_select_font ( resource $printer_handle , resource $font_handle )

A função seleciona uma fonte para desenhar texto. printer_handle deve ser um manipulador válido para uma impressora. font_handle deve ser um manipulador válido para uma fonte.

Exemplo #1 Exemplo printer_select_font()

<?php
$handle = printer_open();
printer_start_doc($handle, "My Document");
printer_start_page($handle);

$font = printer_create_font("Arial", 148, 76, PRINTER_FW_MEDIUM, false, false, false, -50);
printer_select_font($handle, $font);
printer_draw_text($handle, "PHP is simply cool", 40, 40);
printer_delete_font($font);

printer_end_page($handle);
printer_end_doc($handle);
printer_close($handle);
?>
