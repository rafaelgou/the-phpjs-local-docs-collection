odbc_errormsg

(PHP 4 >= 4.0.5, PHP 5)

odbc_errormsg — Obtém a última mensagem de erro

Descrição

string odbc_errormsg ([ resource $connection_id ] )

Retorna uma string contendo a última mensagem de erro do ODBC, ou uma string vazia se não houveram erros.

Parâmetros

connection_id

O identificador de conexão.

Valor Retornado

Se connection_id for especificado, o último estado desta conexão é retornado, senão a última mensagem de qualquer conexão é retornada.

Esta função retorna um valor significante apenas se a ultima consulta odbc falhou (ex. odbc_exec() retornou FALSE).
