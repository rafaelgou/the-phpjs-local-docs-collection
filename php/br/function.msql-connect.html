msql_connect

(PHP 4, PHP 5)

msql_connect — Abre uma conexão mSQL

Descrição

int msql_connect ([ string $hostname [, string $username [, string $password ]]] )

msql_connect() estabelece uma conexão com um servidor mSQL. O parâmetro hostname pode incluir também um número de porta. Ex "hostname:port". O padrão é 'localhost'.

Retorna um identificador de conexão com o mSQL em caso de sucesso, FALSE em caso de erro.

No caso de se fazer uma segunda chamada a msql_connect() com os mesmos argumentos, não é criada uma nova conexão, mas ao invés, a conexão que já esta aberta é retornada.

A conexão com o servidor será fechada tão logo quanto termine a execução do script, a menos que seja fechada antes utilizando msql_close().

Veja também msql_pconnect() e msql_close().
