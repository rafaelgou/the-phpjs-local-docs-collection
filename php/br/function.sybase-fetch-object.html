sybase_fetch_object

(PHP 4, PHP 5)

sybase_fetch_object — Obtém uma linha como um objeto

Descrição

object sybase_fetch_object ( resource $result [, mixed $object ] )

sybase_fetch_object() é similar a sybase_fetch_assoc(), com uma diferença - um objeto é retornado, ao invés de uma matriz.

Em termos de velocidade, a função é idêntica a sybase_fetch_array(), e um pouco mais rápida que sybase_fetch_row() (a diferença é insignificante).

Parâmetros

result

object

Use o segundo parâmetro object para especificar o tipo de objeto que você quer. Se o parâmetro for omitido, o objeto será do tipo stdClass.

Valor Retornado

Retorna um objeto com propriedades que correspondem a linha obtida, ou FALSE se não houver mais linhas.

Changelog

Versão Descrição
4.3.0

Esta função não retornará mais membros númericos de objetos. Funcionamento antigo: 

object(stdclass)(3) {
  [0]=>
  string(3) "foo"
  ["foo"]=>
  string(3) "foo"
  [1]=>
  string(3) "bar"
  ["bar"]=>
  string(3) "bar"
}
Funcionamento novo: 
object(stdclass)(3) {
  ["foo"]=>
  string(3) "foo"
  ["bar"]=>
  string(3) "bar"
}

Exemplos

Exemplo #1 sybase_fetch_object() retornado como Foo

<?php
    class Foo {
        var $foo, $bar, $baz;
    }
    
    // {...]
    $qrh= sybase_query('SELECT foo, bar, baz FROM example');
    $foo= sybase_fetch_object($qrh, 'Foo');
    $bar= sybase_fetch_object($qrh, new Foo());
    // {...]
?>
