connection_status

(PHP 4, PHP 5)

connection_status — Retorna o status da conexão

Descrição

int connection_status ( void )

Veja também connection_aborted() e ignore_user_abort().

Valor Retornado

Retorna o bitfield do status da conexão, que pode ser usado com as constantes CONNECTION_XXX para determinar o status da conexão.

Veja Também

  • connection_aborted() - Verifica se o cliente está desconectado
  • ignore_user_abort() - Estabelece se acontecerá o encerramento do script quando o usuário abortar a conexão
