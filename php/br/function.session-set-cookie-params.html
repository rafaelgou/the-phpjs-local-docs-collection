session_set_cookie_params

(PHP 4, PHP 5)

session_set_cookie_params — Define os parâmetros do cookie de sessão

Descrição

void session_set_cookie_params ( int $lifetime [, string $path [, string $domain [, bool $secure [, bool $httponly ]]]] )

Define parâmetros dos cookies como no arquivo php.ini file. O efeito desta função é apenas pela duração do script. Então, então você precisa chamar session_set_cookie_params() para cada requisição e antes que session_start() seja chamada.

Nota:

O parâmetro secure foi adicionado no PHP 4.0.4, enquanto o parâmetro httponly foi adicionado no PHP 5.2.0.

Veja também as diretivas de configuração session.cookie_lifetime, session.cookie_path, session.cookie_domain, session.cookie_secure, session.cookie_httponly e session_get_cookie_params().
