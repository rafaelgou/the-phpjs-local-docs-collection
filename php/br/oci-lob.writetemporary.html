OCI-Lob::writeTemporary

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob::writeTemporary — Escreve um large object temporário

Descrição

bool OCI-Lob::writeTemporary ( string $data [, int $lob_type = OCI_TEMP_CLOB ] )

Cria um large object temporário e escreve data nele.

Você deve usar OCI-Lob::close quando tiver terminado com esse objeto.

Parâmetros

data

Os dados a escrever.

lob_type

Pode ser um dos seguintes:

  • OCI_TEMP_BLOB é utilizado para cirar BLOBs temporários
  • OCI_TEMP_CLOB é utilizado para criar CLOBs temporários

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.
