gmp_intval

(PHP 4 >= 4.0.4, PHP 5)

gmp_intval — Converte um número GMP para um inteiro

Descrição

int gmp_intval ( resource $gmpnumber )

Esta função permite converter um número GMP para um inteiro.

Parâmetros

gmpnumber

Um número GMP.

Valor Retornado

Um valor integer de gmpnumber.

Exemplos

Exemplo #1 Exemplo gmp_intval()

<?php
// Mostra o valor correto
echo gmp_intval("2147483647") . "\n";
       
// Mostra um resultado errado, acima do limite de intero do PHP
echo gmp_intval("2147483648") . "\n";
       
// mostra o valor correto
echo gmp_strval("2147483648") . "\n";
?>

O exemplo acima irá imprimir:

2147483647
2147483647
2147483648

Notas

Aviso

Esta função retorna um resultado útil somente se o número se adequa ao inteiro PHP (i.e., tipo long com sinal). Se você quer somente imprimir o número GMP, use gmp_strval().
