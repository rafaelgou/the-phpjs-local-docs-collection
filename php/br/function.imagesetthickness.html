imagesetthickness

(PHP 4 >= 4.0.6, PHP 5)

imagesetthickness — Define a espessura para o desenho de linhas

Descrição

bool imagesetthickness ( resource $image , int $thickness )

imagesetthickness() define a espessura das linhas desenhadas quando esta desenhando retangulos, poligonos, elipses, etc. para thickness pixels.

Parâmetros

image

An image resource, returned by one of the image creation functions, such as imagecreatetruecolor().

thickness

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Notas

Nota: This function requires GD 2.0.1 or later (2.0.28 or later is recommended).
