filter_id

(PHP 5 >= 5.2.0)

filter_id — Retorna o ID de um dado nome de filtro

Descrição

int filter_id ( string $filtername )

Parâmetros

filtername

Nome do filtro.

Valor Retornado

ID do filtro em caso de sucesso ou NULL se o filtro não existir.
