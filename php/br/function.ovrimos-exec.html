ovrimos_exec

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_exec — Executa um comando SQL

Descrição

int ovrimos_exec ( int $connection_id , string $query )

ovrimos_exec() executa um comando SQL (query ou update) e retorna um result_id ou FALSE. Evidentemente o comando SQL não deve conter parâmetros.
