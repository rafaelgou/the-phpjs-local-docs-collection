mysqli_disable_reads_from_master

mysqli->disable_reads_from_master()

(PHP 5)

mysqli_disable_reads_from_master -- mysqli->disable_reads_from_master() — Desabilita leitura do master

Descrição

Modo procedural:

bool mysqli_disable_reads_from_master ( mysqli $link )

Modo orientado a objeto (método):

void disable_reads_from_master ( void )
Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Aviso

Esta função tornou-se OBSOLETA e foi REMOVIDA desde o PHP 5.3.0.
