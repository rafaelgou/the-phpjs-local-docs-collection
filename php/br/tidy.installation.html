Instalação

Tidy atualmente é disponível para PHP 4.3.x e PHP 5 como extensão PECL de » http://pecl.php.net/package/tidy.

Nota:

Tidy 1.0 é somente para PHP 4.3.x, e Tidy 2.0 é somente para PHP 5.

Se » PEAR está disponível em seu sistema *nix-like você pode usar o instalador pear para instalar a extensão tidy, pelo seguinte comando: pecl install tidy.

Você sempre pode fazer download do pacote tar.gz e instalar o tidy manualmente:

Exemplo #1 instalação do tidy manualmente no PHP 4.3.x

gunzip tidy-xxx.tgz
tar -xvf tidy-xxx.tar
cd tidy-xxx
phpize
./configure && make && make install

Usuários Windows podem fazer download da extensão DLL de » http://pecl4win.php.net/ext.php/php_tidy.dll.

No PHP 5 você irá precisar somente compilar usando a opção --with-tidy .
