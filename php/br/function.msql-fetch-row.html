msql_fetch_row

(PHP 4, PHP 5)

msql_fetch_row — Retorna uma linha como uma matriz numerada

Descrição

array msql_fetch_row ( resource $query_identifier )

Retorna uma matriz que corresponde a linha obtida, ou FALSE se não houverem mais linhas.

msql_fetch_row() obtem uma linha de dados apartir do resultado de uma query especificado. A linha é retornada como uma matriz. Cada coluna é um índice da matriz, começando em 0.

A próxima chamada a msql_fetch_row() deve retornar a próxima linha do conjunto de resultados, ou FALSE se não houverem mais linhas.

Veja também msql_fetch_array(), msql_fetch_object(), msql_data_seek() e msql_result().
