Transliterator::listIDs

transliterator_list_ids

(PHP >= 5.4.0, PECL intl >= 2.0.0)

Transliterator::listIDs -- transliterator_list_ids — Get transliterator IDs

Descrição

Estilo orientado à objeto

public static array Transliterator::listIDs ( void )

Estilo procedural

array transliterator_list_ids ( void )

Returns an array with the registered transliterator IDs.

Aviso

Esta função não está documentada; somente a lista de argumentos está disponível.

Parâmetros

Esta função não possui parâmetros.

Valor Retornado

An array of registered transliterator IDs on success, ou FALSE em caso de falha.
