shmop_close

(PHP 4 >= 4.0.4, PHP 5)

shmop_close — Fecha um bloco de memória compartilhada

Descrição

void shmop_close ( int $shmid )

shmop_close() é usada para fechar um bloco de memória compartilhada.

Parâmetros

shmid

O identificador de bloco de memória compartilhada criado por shmop_open()

Valor Retornado

Não há valor retornado.

Exemplos

Exemplo #1 Fechando um bloco de memória compartilhada

<?php
shmop_close($shm_id);
?>

Esse exemplo irá fechar o bloco de memória compartilhada identificado por $shm_id.
