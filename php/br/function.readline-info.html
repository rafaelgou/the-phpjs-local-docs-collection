readline_info

(PHP 4, PHP 5)

readline_info — Obtém/Define várias variáveis internas do readline

Descrição

mixed readline_info ([ string $varname [, string $newvalue ]] )

Obtém ou define várias variáveis internas da readline.

Parâmetros

varname

Um nome de variável.

newvalue

Se fornecido, este será o novo valor.

Valor Retornado

Se for utilizada sem parâmetros, esta função retorna uma matriz de valores para as configurações que a readline usa. Os elementos serão indexados pelos seguintes valores: done, end, erase_empty_line, library_version, line_buffer, mark, pending_input, point, prompt, readline_name, e terminal_name.

Se for utilizada com um ou dois parâmetros, o antigo valor é retornado.
