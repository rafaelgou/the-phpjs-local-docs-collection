filepro_rowcount

(PHP 4, PHP 5 <= 5.1.6)

filepro_rowcount — Conta o número de linhas em um banco de dados filePro

Descrição

int filepro_rowcount ( void )

Retorna o número de linhas no banco de dados filePro aberto.

Nota: Quando o safe-mode está ativo, o PHP verifica se os arquivo(s) ou diretórios que estão usados na operação tem o mesmo UID (proprietário) do script que está sendo executado.

Valor Retornado

Retorna o número de linhas no banco de dados filePro aberto, ou FALSE em caso de erro.
