array_reverse

(PHP 4, PHP 5)

array_reverse — Retorna um array com os elementos na ordem inversa

Descrição

array array_reverse ( array $array [, bool $preserve_keys ] )

Recebe como entrada array e retorna um novo array com a ordem dos elementos invertida.

Parâmetros

array

O array de entrada.

preserve_keys

Se definido para TRUE as chaves serão preservadas.

Valor Retornado

Retorna o array invertido.

Changelog

Versão Descrição
4.0.3 O parâmetro preserve_keys foi adicionado.

Exemplos

Exemplo #1 Exemplo da array_reverse()

<?php
$input = array("php", 4.0, array ("verde", "vermelho"));
$result = array_reverse($input);
$result_keyed = array_reverse($input, true);
?>

O programa acima fará com que $result e $result_keyed tenham os mesmos elementos, mas note a diferença entre as chaves. A saída de $result e $result_keyed será: 

Array
(
    [0] => Array
        (
            [0] => verde
            [1] => vermelho
        )

    [1] => 4
    [2] => php
)
Array
(
    [2] => Array
        (
            [0] => verde
            [1] => vermelho
        )

    [1] => 4
    [0] => php
)
