Exemplo #1 Criação de PNG com PHP

<?php

header("Content-type: image/png");
$string = $_GET['text'];
$im     = imagecreatefrompng("images/button1.png");
$orange = imagecolorallocate($im, 220, 210, 60);
$px     = (imagesx($im) - 7.5 * strlen($string)) / 2;
imagestring($im, 3, $px, 9, $string, $orange);
imagepng($im);
imagedestroy($im);

?>
Este exemplo podia ser chamada por uma página com uma tag como: <img src="button.php?text=text">. O script button.php acima então tem esta string "text" e coloca-a por cima da base da imagem que no caso é "images/button1.png" a saída resultante da imagem. Esta é uma forma muito comum para evitar ter que desenhar um nova imagem de botão cada vez que você quer modificar o texto de um botão. Com este método teremos ele dinamicamente gerado.
