odbc_columnprivileges

(PHP 4, PHP 5)

odbc_columnprivileges — Retorna um identificador de resultado que pode ser usado para obter uma lista das colunas e seus privilégios associados

Descrição

resource odbc_columnprivileges ( resource $connection_id , string $qualifier , string $owner , string $table_name , string $column_name )

Lista colunas e os privilégios associados a tabela indicada.

Parâmetros

connection_id

O identificador de conexão.

qualifier

O quantificador.

owner

O proprietário.

table_name

O nome da tabela.

column_name

O argumento column_name aceita padrões de busca ('%' para casar zero ou mais caracteres e '_' para casa um único caractere).

The owner, table_name, and column_name accept search patterns ('%' to match zero or more characters and '_' to match a single character).

Valor Retornado

Retorna um identificador de resultado ODBC ou FALSE em erro. Este identificador de resultado pode ser usado para obter uma lista de colunas e seus privilégios.

O conjunto de resultados tem as seguintes colunas:

  • TABLE_QUALIFIER
  • TABLE_OWNER
  • TABLE_NAME
  • GRANTOR
  • GRANTEE
  • PRIVILEGE
  • IS_GRANTABLE

O conjunto de resultados é ordenado por TABLE_QUALIFIER, TABLE_OWNER e TABLE_NAME.
