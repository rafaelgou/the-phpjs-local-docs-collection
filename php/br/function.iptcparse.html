iptcparse

(PHP 4, PHP 5)

iptcparse — Analisa um bloco binário IPTC em tags simples.

Descrição

array iptcparse ( string $iptcblock )

Analisa um bloco » IPTC em tags simples.

Parâmetros

iptcblock

Um bloco binário IPTC

Valor Retornado

Retorna um array usando a tagmarker como um índice e o valor como o valor. Ela retorna FALSE em erro ou se não há informação IPTC encontrada.
