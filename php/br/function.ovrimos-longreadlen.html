ovrimos_longreadlen

(PHP 4 >= 4.0.3, PHP 5 <= 5.0.5)

ovrimos_longreadlen — Especifica quantos bytes são obtidos dos tipos de dados long

Descrição

bool ovrimos_longreadlen ( int $result_id , int $length )

Especifica quantos bytes serão obtidos dos tipos de dados long (long varchar e long varbinary).

Parâmetros

result_id

Um identificador de resultado, retornado por ovrimos_execute() ou ovrimos_exec().

length

O número de bytes para receber. O padrão é zero.

Valor Retornado

Retorna TRUE.
