mssql_fetch_batch

(PHP 4 >= 4.0.4, PHP 5, PECL odbtp >= 1.1.1)

mssql_fetch_batch — Retorna o próximo lote de registros

Descrição

int mssql_fetch_batch ( resource $result )

Retorna o próximo lote de registros

Parâmetros

result

O resource do resultado que foi avaliado. Este resultado é o que foi retornado ao chamar a mssql_query().

Valor Retornado

Retorna o número do lote como inteiro.

Exemplos

Exemplo #1 Exemplo mssql_fetch_batch()

<?php
$resDb = mssql_connect('localhost', 'user', 'name');
$result = mssql_query('SELECT * FROM MYTABLE', $resDb, 10000);

$intNumRows = mssql_num_rows($result);

while ($intNumRows > 0) {
    while ($arrRow = mssql_fetch_assoc($result)) {
        // Do stuff ...
    }
    $intNumRows = mssql_fetch_batch($result);
}
?>
