sqlite_num_fields

(PHP 5 < 5.4.0, PECL sqlite >= 1.0.0)

sqlite_num_fields — Retorna o número de campos no conjunto de resultados.

Descrição

int sqlite_num_fields ( resource $result )

Retorna o numero de campos no conjunto de resultados result.

Veja também sqlite_column() e sqlite_num_rows().
