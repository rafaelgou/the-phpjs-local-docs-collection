time_nanosleep

(PHP 5)

time_nanosleep — Atrasa para um número de segundos e nanosegundos

Descrição

mixed time_nanosleep ( int $seconds , int $nanoseconds )

Atrasa a execução do programa por um dado número de seconds e nanoseconds.

Parâmetros

seconds

Precisa ser um inteiro positivo.

nanoseconds

Precisa ser um inteiro positivo menor que 1 bilhão.

Valor Retornado

Retorna TRUE em caso de sucesso ou FALSE em caso de falha.

Se o atraso é interrompido por um sinal, um array associativo irá ser retornado com os componentes:

  • seconds - número de segundos restantes do atraso
  • nanoseconds - número de nanosegundos restante do atraso

Exemplos

Exemplo #1 Exemplo da time_nanosleep()

<?php
// Careful! This won't work as expected if an array is returned
if (time_nanosleep(0, 500000000)) {
    echo "Slept for half a second.\n";
}

// This is better:
if (time_nanosleep(0, 500000000) === true) {
    echo "Slept for half a second.\n";
}

// And this is the best:
$nano = time_nanosleep(2, 100000);

if ($nano === true) {
    echo "Slept for 2 seconds, 100 milliseconds.\n";
} elseif ($nano === false) {
    echo "Sleeping failed.\n";
} elseif (is_array($nano)) {
    $seconds = $nano['seconds'];
    $nanoseconds = $nano['nanoseconds'];
    echo "Interrupted by a signal.\n";
    echo "Time remaining: $seconds seconds, $nanoseconds nanoseconds.";
}
?>

Notas

Nota: esta função não é implementada na plataforma Windows
