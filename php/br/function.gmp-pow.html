gmp_pow

(PHP 4 >= 4.0.4, PHP 5)

gmp_pow — Potenciação

Descrição

resource gmp_pow ( resource $base , int $exp )

Eleva base na potência exp.

Parâmetros

base

O número da base.

Ele pode ser qualquer número GMP resource, ou uma string numérica que é possível convertê-la para um número.

exp

A positiva potência para elevar a base.

Valor Retornado

O novo (elevado) número, como um número GMP. O caso de 0^0 reproduzir 1.

Exemplos

Exemplo #1 Exemplo gmp_pow()

<?php
$pow1 = gmp_pow("2", 31);
echo gmp_strval($pow1) . "\n";
$pow2 = gmp_pow("0", 0);
echo gmp_strval($pow2) . "\n";
$pow3 = gmp_pow("2", -1); // Expoente negativo, gera um warning
echo gmp_strval($pow3) . "\n";
?>

O exemplo acima irá imprimir:

2147483648
1
